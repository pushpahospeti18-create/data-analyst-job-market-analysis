# Data Analyst Job Market Analysis Dashboard

## Project Overview

This project analyzes job market trends using Microsoft Excel. The dataset contains 119 job postings and was cleaned, transformed, and analyzed to derive business insights.

## Objectives

* Clean and standardize raw job market data
* Handle missing values and inconsistencies
* Perform exploratory data analysis (EDA)
* Create Pivot Tables and KPI metrics
* Build an interactive Excel dashboard

## Tools Used

* Microsoft Excel
* Pivot Tables
* Pivot Charts
* COUNTIF
* TRIM
* PROPER
* Data Cleaning Techniques
 ## Dataset Note

This project uses a sample job market dataset created for learning purposes and dashboard development. The analysis demonstrates data cleaning, exploratory data analysis (EDA), Pivot Tables, KPI creation, and data visualization techniques in Microsoft Excel.


## Data Cleaning Steps

* Removed duplicate records
* Handled missing values
* Standardized company names
* Standardized location names
* Standardized date formats
* Created skill frequency analysis

## Key Insights

1. **DATASPHERE** was the top hiring company.
2. **Bengaluru** had the highest hiring demand.
3. **AIWORKS** offered the highest average salary (**7.89 LPA**).
4. **63.9%** of job opportunities were suitable for freshers.
5. **Python** emerged as the most in-demand skill.

## Dashboard Components

* Total Jobs KPI
* Top Hiring Company KPI
* Top Hiring City KPI
* Highest Average Salary KPI
* Fresher-Friendly Jobs KPI
